# n=32 Circle Packing In The Unit Square

Goal: place 32 non-overlapping circles inside the unit square `[0, 1]^2` while
maximizing the sum of radii.

## Result

| Quantity | Value |
|---|---:|
| Sum of radii | 2.939572771 |
| Reference | 2.93957 |
| Reference source | Berthold et al., Jan 2026, arXiv:2601.05943 |
| Status | Matches Jan-2026 SOTA at floored 5-decimal precision |
